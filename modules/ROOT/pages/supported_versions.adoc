= Supported Versions

Generally speaking, AeroGear Mobile Services aim to support the latest versions of tools and technologies.
As of July 2019, the following versions are tested and supported:

* Cordova: version 9
* Nodejs: version 10 and 12 LTS
* Android SDK: API level 29
* XCode: version 10.2
