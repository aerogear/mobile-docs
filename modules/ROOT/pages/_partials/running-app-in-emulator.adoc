.Android

****

To run an Android app in an emulator, refer to the following link:https://developer.android.com/training/basics/firstapp/running-app[documentation].

****

[role="secondary"]
.iOS

****

To run an iOS app in an emulator, refer to the following link:https://help.apple.com/xcode/mac/current/#/dev2809b6cff[documentation].

****

[role="secondary"]
.Cordova

****

Cordova cross-platform applications can be run in the following emulators:

* link:https://cordova.apache.org/docs/en/2.7.0/guide/command-line/#android[Android]
* link:https://cordova.apache.org/docs/en/2.7.0/guide/command-line/#ios[iOS]

+
NOTE: Running an iOS application requires running an emulator on macOS

****

[role="secondary"]
.Xamarin

****

Xamarin cross-platform applications can be run in the following emulators:

* link:https://blogs.msdn.microsoft.com/visualstudio/2018/05/08/hyper-v-android-emulator-support/[Android]
* link:https://developer.xamarin.com/getting-started-ios/[iOS]

+
NOTE: Running an iOS application requires running an emulator on macOS

****
