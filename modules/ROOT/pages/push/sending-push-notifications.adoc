= Sending Push Notifications

The {push-ui} allows you to send a Push Notification to devices.

== Prerequisites

[x] Access to the {push-ui}.

== Procedure

. Open a PushApplication and click on the "Send Notification" button. The _Send Push_ dialog is displayed.

. In the +Send Push+ dialog, enter text in the +Message+ text field. 

. Click the +Send Push Notification+ button.

NOTE: It is possible to filter the list of receivers, using +Alias+, +Device Types+ and +Category+. 

Your message is delivered by the 3rd party Push Network to your device.
